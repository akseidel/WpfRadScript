# HDRViewer

File        | Purpose
------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
hdrview.exe | A Radiance PIC file viewer circa 2000. Hdrview.exe may have been written by Haibin Huang while a student at University of California Berkeley. **WpfRadScript** passes the **Radiance PIC** file resulting from **PComb** to **Hdrview.exe** for viewing.
